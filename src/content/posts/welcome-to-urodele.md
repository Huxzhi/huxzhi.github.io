# Welcome to Urodele

This is a sample blog post using Astro Content Collections.

Urodele is a GitHub-based CMS for your Astro blog. Now powered by Content Collections for better Markdown support!

## Features

- Write in pure Markdown
- Edit with CodeMirror
- Render with Astro's built-in Markdown processor
- Full TypeScript support
