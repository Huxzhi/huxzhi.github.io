## 装饰器工厂

可以传参数，用到装饰器工厂，也就是 [柯里化（Currying）](https://zh.javascript.info/currying-partials)

柯里化是一种函数的转换，它是指将一个函数从可调用的  `f(a, b, c)`  转换为可调用的  `f(a)(b)(c)`。

柯里化不会调用函数。它只是对函数进行转换。
