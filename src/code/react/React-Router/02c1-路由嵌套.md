# 02d-路由嵌套

```jsx
const About = (props) => {
    // 获取 match 并解构，获取路径
    const {path} = useRouteMatch();
    
    return (<div>
        <Route path={`${path}/hello`}>
            <Hello/>
        </Route>
    </div>
        );
};
```
