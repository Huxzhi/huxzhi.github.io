# 21-子组件接受父组件

## props.children 表示组件的标签体

子组件接受 父组件的 标签体
子组件接受 父组件的 `class`

```jsx
const Card = (props) => {


    /*
    *   props.children 表示组件的标签体
    * */
    // console.log(props.children);
    return <div className={`card ${props.className}`}>{props.children}</div>;
};

export default Card;
```
