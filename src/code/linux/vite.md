- 开箱即用
- 基于 `vite` 可以快速热更新

### 基于字符串替换而不是 `AST` 替换

`dev` 模式可以正常显示， `build` 模式不可以

所以不要出现如下的代码。 ` `包裹是不起作用，需要 ``` 整个代码块包裹

```
